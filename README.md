# Aizzat Suhardi - Portfolio

Production-grade portfolio website.

**Live:** https://zrrtcs.github.io/portfolio/

## Stack

HTML5 • CSS3 • Alpine.js

## Run Locally

```bash
python3 -m http.server 8000
```

## Deploy

```bash
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
git branch -M main
git push -u origin main
```

Enable GitHub Pages in settings (main branch).

## Update Content

- Edit `index.html` for text/projects
- Edit `css/style.css` colors
- Add CV to `assets/cv.pdf`
